# 🗄️ MySQL Project using Python

## 📌 Project Description

This project demonstrates how to connect **Python with MySQL database** using `mysql-connector-python`.
It includes basic database operations such as creating a database, creating tables, and performing CRUD operations.

This project is designed for beginners to understand how Python interacts with databases.

---

## 🚀 Features

* Connect Python to MySQL server
* Create a new database
* Create tables
* Insert data into tables
* Retrieve data from tables
* Basic CRUD operations (Create, Read, Update, Delete)

---

## 🛠️ Technologies Used

* Python 🐍
* MySQL 🗄️
* mysql-connector-python
* Jupyter Notebook

---

## 📂 Project Files

```
mysql.ipynb   # Main project notebook
README.md     # Project documentation
```

## 🎯 Learning Outcomes

* Understanding of database concepts
* Hands-on experience with MySQL
* Python-MySQL integration
* Writing basic SQL queries
